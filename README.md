# TextToBraille
This is my Bachelor's Thesis project - Change any text into tactile Braille output

# How it works
It takes scanned text from your Android device, from an app and /
sends it to ESP32 via bluetooth connection

*App is designed specifically for Android devices*

# What's in this repo and how to find it
This repo consists of two parts:
- an Android Studio app
- code for text to Braille output translation (for ESP32)

# Which ESP32 I used
I used *uPesy ESP32 Wroom DevKit*
